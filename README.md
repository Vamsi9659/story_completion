# Story Completion

Story Completion is a full-stack web application for students and teachers to interact with story-writing prompts and analyze student responses.

## Features
- Student login and signup
- Teacher login and signup
- Story prompt generation
- Story submission and analysis
- Teacher report viewing

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI
- Database: MongoDB

## Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Project Structure
- backend/ - FastAPI backend services
- frontend/ - React frontend application
